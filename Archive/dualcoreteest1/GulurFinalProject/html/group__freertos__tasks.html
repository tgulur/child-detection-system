void motionDetectionTask ( void * pvParameters)

Motion detection and persistence tracking task.

Handles PIR sensor interrupts, maintains motion persistence counter using Hardware Timer 0 for 6-second updates, and manages emergency stop functionality. Implements temperature-adjusted alert level calculation and system state management.

Parameters
pvParametersUnused FreeRTOS parameter
Note
Runs at 50Hz (20ms period) on Core 0
Uses Hardware Timer 0 for precise 6-second counter updates
Implements counter-based persistence to handle PIR sensor limitations

void temperatureTask ( void * pvParameters)

Temperature and humidity monitoring task.

Reads AM2302 sensor data every 3 seconds and processes temperature-based threshold calculations at 64Hz. Updates global temperature variables and sends data to other tasks via temperature queue.

Parameters
pvParametersUnused FreeRTOS parameter
Note
Runs at 64Hz (15ms period) on Core 0
Sensor reading limited to 3-second intervals due to hardware constraints

Enhanced motion detection and processing task (50Hz, Core 0) 

This is the core task that handles PIR motion detection, maintains the motion persistence counter, processes emergency stop button input, and coordinates the overall system response. It implements a sophisticated counter-\/based approach to handle PIR sensor timeout limitations.

Key Responsibilities\+:
\begin{DoxyItemize}
\item Process PIR sensor interrupts and maintain motion persistence counter
\item Handle emergency stop button functionality
\item Implement 6-\/second polling cycle for counter decrement
\item Calculate temperature-\/adjusted alert levels
\item Control status LED based on motion state
\item Send motion data to display and alert tasks
\end{DoxyItemize}

Task Specifications\+:
\begin{DoxyItemize}
\item Frequency\+: 50Hz (20ms period)
\item Core\+: 0 (sensor processing core)
\item Priority\+: 2
\item Stack\+: 4096 bytes
\end{DoxyItemize}
